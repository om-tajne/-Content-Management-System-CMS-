# 🚀 Custom Media Content Management System (CMS)

A full-stack, secure platform for managing digital content and user authentication.

## 🛠️ Tech Stack
- **Backend:** Node.js, Express.js
- **Frontend:** EJS (Embedded JavaScript), CSS3, Bootstrap
- **Database:** PostgreSQL
- **Security:** Bcryptjs (Password Hashing), Express-Session (Authentication)

## 📁 Key Features
- **User Authentication:** Secure Sign-up and Login system.
- **Admin Dashboard:** Centralized interface to manage media and posts.
- **MVC Architecture:** Highly organized code for scalability.
- **Session Management:** Persists user login status securely.

## ⚙️ Setup Instructions
1. Install dependencies: `npm install`
2. Configure your `.env` with your DB details.
3. Start the server: `node app.js`
